# 接口文档


## 用户模块
### 添加用户
+ URL:```/user```
+ 方法：```POST```
+ 参数：
|参数|描述|
|userName|用户名|

+ 返回值：

    ```JSON
    {
        "result":'true'
    }
    ```

### 删除用户
+ URL：```/user```
+ 方法：```DELETE```
+ 参数：
|参数|描述|
|userId|用户ID|

+ 返回值：

    ```JSON
    {
        "result":'true'
    }
    ```


### 查询用户
+ URL:```/user```
+ 方法：```GET```
+ 参数：
|参数|描述|
|userId|用户ID|

+ 返回值：

    ```JSON
    {
        "result":'true'
    }
    ```
